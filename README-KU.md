# زیرەکی ئۆفلاین — وەشانی Text-only 2GB + Smart Answer Fix

ئەم پرۆژەیە وێبسایتێکی کوردی/RTL ـە بۆ چاتکردن لەگەڵ زیرەکییەکی دەستکرد کە لە ناو براوزەر کار دەکات. بەڕێوەبەر دەتوانێت زانیاریی تایبەتی فێری بکات، هەروەها Knowledge Pack ـی گشتی بە پارچەی JSONL لە IndexedDB هەڵدەگیرێت تا دواتر بە تەواوی ئۆفلاین سودی لێ ببینرێت.

## گۆڕانکاریی گرنگی ئەم وەشانە

- چارەسەری وەڵامی هەڵە بۆ پرسیارە بنەڕەتییەکان کرا. ئێستا «ئاو چیە» و «ژیان چیە» ڕاستەوخۆ لە پێناسەی بنەڕەتی وەڵام دەدرێنەوە، نەک لە هەواڵ یان دەقی نامەربووت.
- بۆ پرسیاری کورت و پێناسەیی، ئەنجامی بەڕێوەبەر تەنها ئەگەر ناونیشان/پرسیار/alias ـی ڕاستەوخۆی هەمان بابەت بێت پێشەنگی وەردەگرێت.
- پەکی ٢GB بە شێوەی **تەنها تێکست** دیاری کرا: JSONL، بێ وێنە، بێ ڤیدیۆ، بێ HTML، بێ فایلە باینەرییەکان.
- ئامرازی نوێ زیاد کرا: `tools/build-web-knowledge-pack.js` بۆ دروستکردنی Knowledge Pack ـی تێکست لە Wikipedia plaintext extracts.
- پەکی ناوخۆیی هێشتا fallback ـی بچووکە، نەک پەکی ڕاستەقینەی ٢GB.

## کردنەوەی وێبسایت

لە ناو فۆڵدەری پرۆژەکە:

```bash
node tools/server.js 8080
```

پاشان بکەرەوە:

```text
http://localhost:8080
```

`localhost` باشترینە چونکە Service Worker، fetch و IndexedDB بە دروستی کار دەکەن. بە `file://` زۆرجار fetch ـی manifest/JSONL لە براوزەر قەدەغە دەبێت.

## جیاوازی پەکی ناوخۆیی و پەکی ٢GB

پەکی ناوخۆیی تەنها بۆ ئەوەیە کە سیستەم لە یەکەم تاقیکردنەوەدا بەتاڵ نەبێت. ئەو پەکە بچووکە و ٢GB نییە.

پەکی ٢GB باید لە سێرڤەر یان localhost دابین بکرێت. قەبارەکە زۆر گەورەیە، بۆیە لە ZIP ـی ئەم پرۆژەیەدا دانەنراوە. ئەم پرۆژەیە ئامادەیە کە پەکی ٢GB ـی تێکست دابگرێت و لە IndexedDB هەڵیبگرێت.

## دروستکردنی پەکی ٢GBی تێکست لە ئینتەرنێت

ئەم فرمانە Wikipedia ـی کوردی/عەرەبی/ئینگلیزی بەکار دەهێنێت و تەنها تێکست دەهێنێت:

```bash
node tools/build-web-knowledge-pack.js knowledge-pack 2 ckb,ar,en 104857600
```

مانای پارامێتەرەکان:

- `knowledge-pack`: فۆڵدەری دەرچوون.
- `2`: ئامانجی ٢GB.
- `ckb,ar,en`: زمانەکان.
- `104857600`: قەبارەی هەر پارچە، واتە 100MB.

ئەم ئامرازە فایلەکانی خوارەوە دروست دەکات:

```text
knowledge-pack/manifest.json
knowledge-pack/chunk-0001.jsonl
knowledge-pack/chunk-0002.jsonl
...
```

پاشان وێبسایتەکە لە `localhost` بکەرەوە و manifest ـەکە دابنێ:

```text
knowledge-pack/manifest.json
```

دوگمەی **دابەزاندن/نوێکردنەوە** یان **فێربوون لە ئینتەرنێت - تێکست** بەکاربهێنە.

## دروستکردنی پەک لە فایلی JSONLی خۆت

ئەگەر خۆت فایلی `input.jsonl` ـت هەیە:

```bash
node tools/make-knowledge-pack.js input.jsonl knowledge-pack "General Life Knowledge 2GB Text Only" 104857600
```

هەر دێڕێکی JSONL دەبێت شێوەیەکی وەک ئەمەی هەبێت:

```json
{"id":"doc-1","title":"ناونیشان","category":"بەش","question":"پرسیار؟","aliases":["پرسیاری هاوشێوە"],"answer":"وەڵام","content":"وردەکاری تێکست","tags":["tag1","tag2"],"priority":1}
```

## پشکنینی پەک

```bash
node tools/check-pack.js knowledge-pack/manifest.json
```

ئەگەر هەڵەی SHA-256، JSONL یان ڕێڕەو هەبوو، ئەم ئامرازە دەریدەخات.

## CORS بۆ سێرڤەری دەرەکی

ئەگەر وێبسایتەکە لە domain ـێکە و Knowledge Pack لە domain ـێکی ترە، سێرڤەری Knowledge Pack پێویستە ئەم header ـانە بنێرێت:

```http
Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, HEAD, OPTIONS
Access-Control-Allow-Headers: Content-Type, Range
Accept-Ranges: bytes
Content-Type: application/x-ndjson; charset=utf-8
```

ئەگەر ناتەوێت CORS ڕێکبخەیت، Knowledge Pack لە هەمان domain/فۆڵدەری وێبسایت دابنێ.

## چۆن تاقی بکەیتەوە کە وەڵامەکان چاک بوون؟

لە چاتدا ئەمانە بنووسە:

```text
ئاو چیە
ژیان چیە
مرۆڤ چیە
هەوا چیە
```

دەبێت وەڵامەکان لە پێناسەی بنەڕەتی بێن، نەک لە دەقی هەواڵ/وەرزش/بابەتی نامەربووت.

## تێبینی گرنگ

ئەم سیستەمە LLM ـی گەورەی وەک ChatGPT نییە و بەبێ API کار دەکات. بۆ ٢GB، سیستەمەکە تێکست زۆر هەڵدەگرێت و بە گەڕانی زیرەک و پێشەنگیی پێناسەکان وەڵام دەداتەوە. بۆ بابەتە هەستیارەکان وەک پزیشکی، یاسایی و دارایی، وەڵامەکان ڕێنمایی گشتین و جێگای شارەزای پیشەیی ناگرنەوە.
